Projet E6

GestTravaux

Application de gestion de travaux immobiliers pour la société fictive IMMOSYNC. Projet en binôme couvrant tout le cycle : création de chantiers, gestion de devis, suivi d'avancement et comparaison d'offres d'entrepreneurs.

Mon rôle

  • Développement de l'application desktop Java (JavaFX + Hibernate)
  • Conception des entités JPA et du schéma MySQL (14 tables)
  • Système d'authentification et gestion de session
  • CRUD complet : propriétaires, biens, chantiers, prestations
  • Navigation entre vues avec SceneRouter

Partie Web (Symfony 8)

  • Interfaces web pour inspecteurs, entrepreneurs et propriétaires
  • Doctrine ORM avec 11 entités mappées
  • Sécurité multi-rôles et gestion de documents
  • Docker Compose pour l'environnement de développement
Java JavaFX Hibernate MySQL Symfony 8 PHP 8.4 Doctrine Docker Maven
